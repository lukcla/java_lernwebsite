# Prog2-Lernwebsite – Projektpaket

## Schnellstart in VS Code
1. Diesen Ordner in VS Code öffnen (Datei → Ordner öffnen).
2. Claude-Code-Erweiterung installieren und anmelden.
3. Beispiel-Auftrag an Claude:
   „Befülle Kapitel 6 (Exceptions) der Website anhand von skripte/teil1/06-exceptions.md –
   beachte die Regeln in CLAUDE.md."
4. Website per Doppelklick auf `prog2-teil1-lernwebsite.html` im Browser ansehen.

## Struktur
- `prog2-teil1-lernwebsite.html` – Lernwebsite zu Teil 1 (Gerüst, wird kapitelweise befüllt)
- `skripte/teil1/` – Skript Teil 1, in Kapitel zerlegt (9 Dateien)
- `skripte/teil2/` – Skript Teil 2, in Kapitel zerlegt (11 Dateien; Generics in 03a–03e)
- `skripte/INHALTSVERZEICHNIS.md` – Gesamtübersicht
- `CLAUDE.md` – Arbeitsregeln für Claude (Quellenpflicht, Formatierung etc.)

## Hinweis zu Teil 2
Für Teil 2 existiert noch kein Website-Gerüst. Sag Claude bei Bedarf:
„Erstelle analog zu Teil 1 ein Website-Gerüst für Teil 2 anhand von skripte/teil2/INHALTSVERZEICHNIS.md."
